## Mailpit

![Mailpit](https://img.shields.io/badge/Mailpit-F36F21?logo=maildotru&logoColor=white)

**Email and SMTP testing tool**

Mailpit is a lightweight email testing tool that catches all outgoing emails for inspection. It provides a web UI to browse captured messages and an SMTP server for local testing. Features include:
- Web UI for browsing captured emails
- SMTP server on port 1025
- REST API for automated testing
- HTML and plain text message preview
- Attachment support
- Message search and filtering

| Setting | Value |
|---------|-------|
| Default Port | `8025` |
| Suggested Subdomain | `mailpit` |
| Public Access | No (email testing data) |
| Website | [mailpit.axllent.org](https://mailpit.axllent.org) |
| Source | [GitHub](https://github.com/axllent/mailpit) |
